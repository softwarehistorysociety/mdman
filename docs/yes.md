YES(1) - General Commands Manual

# NAME

**yes** - be repetitively affirmative

# SYNOPSIS

**yes**
\[*expletive*]

# DESCRIPTION

The
**yes**
utility outputs
*expletive*,
or, by default,
"y",
forever.

# SEE ALSO

jot(1),
seq(1)

The
**yes**
command appeared in
Version&#160;7 AT&T UNIX.
