AFCONVERT(1) - General Commands Manual

# NAME

**afconvert** - Audio File Convert

# SYNOPSIS

**afconvert**
\[*option*]
*src\_audiofile*
*dest\_audiofile*

# DESCRIPTION

Audio File Convert will convert a source audio file to a new audio file with the specified file and data types
